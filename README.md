# Thiruppali App - Daily Mass Readings & Songs

A Progressive Web Application (PWA) designed to provide daily Tamil Mass readings, reflections, saint histories, and Christian songs for the faithful.

### Prerequisites

-   Node.js (v18 or higher)
-   npm (v9 or higher)

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/ajf013/thiruppali-app.git
    cd thiruppali-app
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Building for production:
    ```bash
    npm run build
    ```
